## ⚠️ 当前限制

### 1. Windows端未集成驱动
**现状**: Windows客户端只能接收URB响应，无法实际使用USB设备  
**原因**: 需要整合usbip-win2的VHCI驱动  
**影响**: 设备导入后Windows看不到虚拟USB设备

### 2. OUT传输数据未传递
**现状**: Bulk OUT传输发送空数据  
**原因**: 需要从payload中提取实际数据  
**修复**: 修改`executeBulkTransfer()`读取后续数据

### 3. 等时传输不支持
**现状**: ISO传输返回错误  
**原因**: Android USB Host API不完全支持ISO  
**备选**: 可能需要JNI调用底层API

### 4. 无异步URB队列
**现状**: URB同步执行，可能阻塞  
**影响**: 高并发场景性能下降  
**优化**: 使用Coroutine或ThreadPool
