## 🎯 P0级问题（必须修复）

### 1. OUT传输数据丢失
**位置**: UrbHandler.kt + UsbService.kt  
**问题**: Bulk/Interrupt OUT传输时，实际数据未从socket读取  
**影响**: 所有写入设备的操作失败  
**修复方案**:
```kotlin
// UsbService.kt - handleUrbSubmit
val urbSubmit = UrbSubmit.parse(urbPayload)

// 读取后续的数据payload
val dataPayload = if (urbSubmit.dataLen > 0 && urbSubmit.direction == URB_DIR_OUT) {
    ByteArray(urbSubmit.dataLen).also { 
        var read = 0
        while (read < it.size) {
            val n = input.read(it, read, it.size - read)
            if (n < 0) break
            read += n
        }
    }
} else byteArrayOf()

// 传递给handler
val urbComplete = urbHandler.handleUrbSubmit(urbSubmit, dataPayload)
```

```kotlin
// UrbHandler.kt - handleUrbSubmit签名修改
fun handleUrbSubmit(urbSubmit: UrbSubmit, dataPayload: ByteArray = byteArrayOf()): UrbComplete

// executeBulkTransfer中使用dataPayload
if (urbSubmit.direction == URB_DIR_OUT) {
    val result = conn.bulkTransfer(endpoint, dataPayload, dataPayload.size, USB_TIMEOUT)
    ...
}
```
