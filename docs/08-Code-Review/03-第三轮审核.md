## 已知限制

### 当前版本限制

1. **USB设备虚拟化未完成**
   - 原因: 需要usbip-win2驱动集成
   - 影响: 无法在Windows上实际使用USB设备
   - 状态: 框架已完成，等待驱动开发

2. **URB数据传输未实现**
   - 原因: 需要完整的USB/IP协议栈
   - 影响: 无法读写USB设备数据
   - 状态: 协议预留，待后续实现

3. **安全性不足**
   - 无认证机制
   - 明文传输
   - 建议: 生产环境前必须添加

### 技术债务

1. **线程安全** - 建议使用ConcurrentHashMap
2. **资源清理** - 设备移除时清理importedDevices
3. **Socket超时** - 防止永久阻塞
4. **错误提示** - 可以更用户友好
