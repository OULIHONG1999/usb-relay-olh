## 新发现的问题

### 🟡 中等问题

#### 1. importedDevices线程安全
**位置**: UsbService.kt  
**问题**: importedDevices和deviceIdMap在多线程环境下未同步  
**影响**: 可能导致数据不一致  
**建议**: 使用ConcurrentHashMap或synchronized块

```kotlin
import java.util.concurrent.ConcurrentHashMap

private val importedDevices = ConcurrentHashMap<Int, UsbDevice>()
private val deviceIdMap = ConcurrentHashMap<Int, String>()
```

#### 2. 缺少导入设备的清理
**位置**: UsbService.kt  
**问题**: 设备移除时没有清理importedDevices  
**建议**: 在handleDeviceDetached中清理

```kotlin
fun handleDeviceDetached(deviceName: String): Boolean {
    availableDevices.remove(deviceName)
    
    // Clean up imported devices
    val device = availableDevices[deviceName]
    if (device != null) {
        importedDevices.entries.removeAll { it.value == device }
    }
    
    broadcastDeviceUpdate()
    return false
}
```
