# 电脑端（Windows）验证指南

## 方案概述

你可以选择使用两个方案中的一个：
1. **方案一：使用 usbip-win2（推荐，成熟稳定）** - 完整标准 USB/IP 协议，WHQL 驱动
2. **方案二：使用 relay-client（本项目原始客户端）** - 自定义协议，需要配合自定义的 Android 服务器

---

## 方案一：使用 usbip-win2（推荐）

### 1. 准备工作

#### 系统要求
- Windows 10 x64 1903 (OS build 18362) 或更高
- Windows 11 ARM64
- Windows 11 x64

#### 安全准备（重要）
**在安装之前必须创建系统还原点！**
1. 在任务栏搜索框输入 "创建还原点" 并打开
2. 选择系统驱动器（C: 盘）
3. 点击 "创建"
4. 输入描述（例如 "USBip 安装前"）
5. 点击 "创建" 等待完成

---

### 2. 下载 usbip-win2

#### 方式一：手动下载
1. 访问最新发布页面：
   - https://github.com/vadimgrn/usbip-win2/releases/latest

2. 下载最新安装包（当前 v0.9.7.7）：
   - 文件名：`USBip-Setup-v0.9.7.7.exe`

3. 保存到：
   - `E:\WORK\usb-relay-olh\usbip-win2-download\`

---

### 3. 安装 usbip-win2

#### 步骤一：测试签名模式（需要吗？）
- 查看 README：
  - 最新版本（v0.9.7.0+）是 WHLK 认证的，可能不需要测试签名模式
- 如果安装失败，可能需要启用：
  ```powershell
  # 以管理员身份运行 PowerShell
  bcdedit.exe /set testsigning on
  # 重启电脑
  ```

#### 步骤二：安装
- 以管理员身份运行 `USBip-Setup-v0.9.7.7.exe`
- 按照安装向导操作
- ⚠️ 注意：安装过程中，所有 USB 设备会暂时重启！

---

### 4. 验证安装

#### 验证命令
打开命令提示符或 PowerShell（普通用户即可）：
```powershell
# 查看版本
usbip.exe --help

# 如果安装成功，会显示帮助信息
```

---

### 5. 基本使用方法

#### 查看远程设备列表
```powershell
usbip.exe list -r <Android 设备 IP 地址>
```

#### 附加远程 USB 设备
```powershell
# 先列出设备，找到 busid
usbip.exe attach -r <IP> -b <busid>
```

#### 查看已附加的设备
```powershell
usbip.exe port
```

#### 分离设备
```powershell
usbip.exe detach -p <端口号>
# 或者分离所有
usbip.exe detach -all
```

## 方案二：使用 relay-client（本项目原始客户端）

### 1. 验证已编译的程序

relay-client 已经编译成功！

#### 位置
```
E:\WORK\usb-relay-olh\relay-client\windows\build\relay-client.exe
```

#### 测试运行
```powershell
cd E:\WORK\usb-relay-olh\relay-client\windows\build
.\relay-client.exe
```
应该会显示：
```
Relay Client v0.1.0
Usage: relay-client.exe <host> [port]
```

---

### 2. 重新编译（如果需要）

#### 依赖
- CMake
- MinGW GCC

#### 编译命令
```powershell
cd E:\WORK\usb-relay-olh\relay-client\windows
mkdir build  # 如果不存在
cd build
cmake .. -G "MinGW Makefiles"
mingw32-make
```

---

### 3. relay-client 限制

⚠️ **重要**：
- relay-client 使用的是自定义协议，**不兼容标准 USB/IP 协议**
- 需要配合同样实现自定义协议的 Android 服务器使用
- 如果你想使用标准协议，使用方案一（usbip-win2）
