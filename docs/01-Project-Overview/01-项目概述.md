# USB Relay OLH - 项目概述

## 项目目标
通过网络将 Android 设备连接的 USB 设备共享给 Windows 电脑使用，实现 USB 设备的远程访问。

## 项目架构

```
物理 USB 设备
    ↓ (USB OTG)
Android 设备 (USB/IP 服务器)
    ↓ (TCP/IP 网络, 端口 3240)
Windows 电脑 (USB/IP 客户端 + 虚拟 USB 驱动)
    ↓
Windows 应用程序 → 识别为本地 USB 设备
```

## 推荐方案

基于研究结果，推荐使用以下成熟的开源方案：

### 服务器端 (Android)
**项目**: [Android-Usbipdcpp](https://github.com/yunsmall/Android-Usbipdcpp)
- 语言: Kotlin + C++ (NDK)
- 特性: 完整 USB/IP 协议实现，多设备支持，可配置端口
- 许可证: GPL-3.0

### 客户端 (Windows)
**项目**: [usbip-win2](https://github.com/vadimgrn/usbip-win2)
- 语言: C++ (驱动) + C++ (用户空间)
- 特性: WHLK 认证驱动，完整 USB/IP 协议，零拷贝传输
- 许可证: BSD-2-Clause
