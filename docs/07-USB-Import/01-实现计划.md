### 3. 协议扩展

#### 新增命令
- CMD_IMPORT_DEVICE (0x1003): 客户端请求导入设备
- CMD_IMPORT_RES (0x0004): 服务器响应导入结果

#### IMPORT_DEVICE Payload
```
Offset  Size  Field       Description
------  ----  ----------  ---------------------------
0       2     dev_id      要导入的设备ID
2       2     reserved    保留
```

#### IMPORT_RES Payload
```
Offset  Size  Field       Description
------  ----  ----------  ---------------------------
0       2     status      0=成功, 非0=失败
2       2     dev_id      设备ID
4       4     result_code 详细结果码
8       N     message     错误消息（可选）
```
